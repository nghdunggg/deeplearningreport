1. Exploratory Data Analysis (EDA)

The CIFAR-10 dataset consists of 60,000 32x32 color images in 10 classes, with 6,000 images per class. There are 50,000 training images and 10,000 test images.

Example classes: plane, car, bird, cat, deer, dog, frog, horse, ship, truck.

2. Model Architecture

For this assignment, we implemented a Convolutional Neural Network (CNN) designed to extract spatial features from the 32x32 RGB images. The pipeline includes:

  • Data Augmentation: Normalization and tensor conversion.
  • Layers: Sequential Convolutional layers followed by Max-Pooling and Fully Connected (Linear) layers.
  • Optimizer: SGD or Adam was utilized to minimize the Cross-Entropy loss.

3. Training Performance

The model was trained on a Tesla T4 GPU using PyTorch. Below is a summary of the performance:

Metric Value
Training Samples 50,000
Test Samples 10,000
Batch Size 16 (EDA) / Configurable for Training

4. Conclusion & Observations

By leveraging deep convolutional layers, the model learns to identify complex patterns such as textures and shapes. Key challenges observed included distinguishing between visually similar classes like "cat" and "dog."
